Welcome to the FillOut API (now with filters!)

You can access the only available api endpoint at: {domain}/:formId/filteredResponses. Here are some example URLs:

1. using limit and sort:

{domain}/cLZojxk94ous/filteredResponses?limit=5&sort=asc

2. using a filter for "yearly checkin" date equal to 2024-02-01 ({"id":"dSRAe3hygqVwTpPK69p5td","condition":"equals","value":"2024-02-01"}):

{domain}/cLZojxk94ous/filteredResponses?filters=%5B%7B%22id%22%3A%22dSRAe3hygqVwTpPK69p5td%22%2C%22condition%22%3A%22equals%22%2C%22value%22%3A%222024-02-01%22%7D%5D

3. using two filters for "yearly checkin" date less than to 2024-02-02 and "anything else to share" equal to "Nothing much to share yet!":

{domain}/cLZojxk94ous/filteredResponses?filters=%5B%7B%22id%22%3A%22dSRAe3hygqVwTpPK69p5td%22%2C%22condition%22%3A%20%22less_than%22%2C%22value%22%3A%222024-02-03%22%7D%2C%2520%7B%22id%22%3A%224KC356y4M6W8jHPKx9QfEy%22%2C%22condition%22%3A%20%22equals%22%2C%22value%22%3A%22Nothing%20much%20to%20share%20yet!%22%7D%5D
